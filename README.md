# FnSolution Consulting Website

A modern, premium corporate consulting website built with Next.js, featuring stunning animations and a sophisticated design.

## About FnSolution

FnSolution is a leading corporate consulting firm specializing in:
- Business Consulting
- International Expansion
- Service Development
- Leadership Development

**Our Mission**: We Make Your Value Visible.

## Tech Stack

- **Framework**: Next.js 16
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Font**: Pretendard (Korean), Sans-serif

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
